# Frontend Components

Overview of the Streamlit components and UI elements used in the dashboard.

## 🧩 Component Overview

The dashboard is built with modular Streamlit components for better maintainability.

## 🔐 Authentication Components

### Login Form
- Email input field
- Password input field
- Login button
- Error message display

### Token Status Display
- Token validity indicators
- Creation timestamps
- Refresh controls

## 📦 Order Components

### Order Creation Form
- User validation
- Order creation button
- Success/error feedback

### Order List Display
- Tabular order view
- Status indicators
- Action buttons

## 🛒 Item Management Components

### Item Addition Form
- Product name input
- Quantity selector
- Price input
- Add button

### Item List Display
- Item details table
- Remove buttons
- Total calculations

---

**These components work together to create a seamless user experience!** ✨
